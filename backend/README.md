## 🔐 Auth Endpoints

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| POST | `/api/auth/register` | — | Register (name ≥2, email valid, password ≥8) |
| POST | `/api/auth/login`    | — | Login → JWT |
| GET  | `/api/auth/me`       | 🔒 | Current user |
| POST | `/api/auth/logout`   | 🔒 | Logout |
